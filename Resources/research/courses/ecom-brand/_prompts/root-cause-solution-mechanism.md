Two linked 3-prompt chains that turn a problem into advertorial-ready copy: first uncover the real root cause, then build the unique solution mechanism (the "why this works when others fail"). Use when you need a defensible UMP/UMS for a supplement or health angle, for example Goatly bone-health.

## Chain A: Root Cause (run in order)

```text
PROMPT 1 (deep research)
Hi, I want you to act as a world-class direct response copywriter for a $100M/year brand in
the [NICHE]. Research and explain the root cause of a specific problem experienced by our
ideal customer (the [AVATAR]). Not surface-level: uncover what is really happening underneath,
backed by credible scientific and medical research.
Task: 1. Research the root cause of: "[INSERT PROBLEM]". 2. Find what is actually causing it.
3. Ensure the explanation is backed by logical, factual, research-based information.
[Copy the results into a document called ROOT CAUSE DRAFT, then run Prompt 2.]

PROMPT 2 (simplify + advertorial-ready)
Act as a world-class direct response copywriter for a $100M/year brand in the [NICHE]. You are
given a research-based, science-backed document explaining the root cause. Translate the
complexity into clear, emotional, easy-to-understand copy in 3 phases:
Phase 1 - Deep understanding: read the document, find what is actually causing the issue.
Phase 2 - Simplify and visualize: summarize the root cause in plain English (no jargon),
re-explain it like you are talking to a 4th grader, add a real-life analogy or metaphor.
Phase 3 - Copywriting breakdown: write an advertorial-ready section that goes straight into
the root cause (no story), short punchy sentences, easy and entertaining to read, with a
metaphor that makes the cause visual. A plug-and-play "why it happens" paragraph.

PROMPT 3 (UGC clip)
Act as a UGC-focused creative strategist for a $100M/year brand in the [NICHE]. Using the
research above, write a short authentic UGC-style clip where a real woman who personally
struggles with [PROBLEM] casually explains what causes it in her own voice, like a girl talking
to a friend on TikTok. Simple casual language, no jargon, include a visual metaphor if natural.
No story, no product, no intro or outro, just the root cause explanation. Then list 7-10 short
natural talking points creators can riff on.
```

## Chain B: Solution Mechanism (run after Chain A)

```text
PROMPT 1 (develop the mechanism)
Act as a world-class product strategist for a $100M/year brand in the [NICHE]. You are given a
document explaining the root cause. Study it and develop a Unique Solution Mechanism: a clear,
logical theory of action that explains how the problem can be fixed at the root level. This is
NOT about the product, it is the big "aha" insight bridging root cause and solution. If the root
cause has multiple factors, give a specific theory for fixing each. Keep it unique, logical,
directly tied to the root cause, and something the prospect has not heard but immediately makes
sense. Deliver: 1. a clear plain explanation of the overall mechanism, 2. how it solves each
element of the root cause.

PROMPT 2 (plain-language "how it works" copy)
Act as a world-class direct response copywriter and product strategist for a $100M/year brand.
You are given the solution mechanism. Do NOT reference the root cause, only the mechanism.
Phase 1: understand how it works logically/biologically/behaviorally. Phase 2: explain it in
plain but credible language, keep useful technical terms but explain them like a smart friend,
add one clean visual metaphor. Phase 3: write a punchy "How It Works" section, short emotionally
resonant sentences, light clearly-explained science, talking directly to someone tired of false
promises. It should make her say "That actually makes sense, and no one explained it to me
like this before."

PROMPT 3 (UGC solution clip)
Act as a UGC-focused creative strategist. Write the next UGC segment where the creator, having
just explained the root cause, casually explains the solution mechanism ("Okay so here's what
actually fixes it..."). Relaxed, curious but confident, like she just learned it and is putting
her friends on. No product yet, just the theory of action. One simple analogy if it fits. Then
7-10 freestyle talking points for creators.
```
