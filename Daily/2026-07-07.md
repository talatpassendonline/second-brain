## Session log

### [[Goatly-UK]] UK-checkout: Checkify-restant eruit, Lasso erin (commits 677df8d + a4ff326, `uk`)

Talat meldde dat de checkout op [[Goatly-UK]] (trygoatly.com) bij het laden heel even als een Checkify-link flitste. Oorzaak: de theme was gedupliceerd van de NL/BE-store (die via [[Checkify]] draait) en `layout/theme.liquid` laadde nog `pay.checkify.pro/static/js/shopify.js`. Dat script kaapte de checkout-knop, faalde op de UK-store (niet gekoppeld) en viel terug op de native checkout. Die mislukte redirect = de flits.

Eerst Checkify chirurgisch verwijderd (commit 677df8d): storeScript-blok + dode preconnects in `theme.liquid`, plus twee checkify-preconnects in `templates/page.adv2-uk-bones.liquid`. Daarna bleek de UK-checkout eigenlijk via **Lasso** te moeten lopen (custom checkout op eigen domein `checkout.trygoatly.com`), niet de native Shopify-checkout. Het Lasso-script stond alleen in de ongepubliceerde "Lasso"-theme, nooit op live. `lasso-script-v4.min.js` (store-id `6a46cba3f4509ef23682dee0`, checkout-url `checkout.trygoatly.com/checkout`) toegevoegd vlak voor `</body>` (commit a4ff326). Endpoints vooraf gecheckt: script + checkout-root geven beide HTTP 200. Live geverifieerd via Shopify MCP: Checkify weg, Lasso-script aanwezig.

**Twist: live theme is de losse "Lasso"-theme, niet `uk`.** Bleek dat de eigenlijke live/funnel-theme een aparte theme "Lasso" is (id 202877141334), met **UpCart** (cart-drawer) + **Kaching Bundles** app-embeds én het Lasso-checkout-script. Die theme is NIET git-gekoppeld. Talat heeft die theme gepubliceerd; hij had het oude Checkify-storeScript nog in de `<head>` → flits terug. Omdat die theme live (`MAIN`) én niet-git is, kon ik er niet in schrijven (Shopify-koppeling blokkeert live-theme writes). Talat heeft het Checkify-blok zelf in de theme-editor verwijderd. Geverifieerd via MCP: `layout/theme.liquid` volledig schoon, Lasso-script aanwezig, checkout flitst niet meer. Mijn `uk`-commits (677df8d + a4ff326) staan nu op de ongepubliceerde `trygoatly/uk`-theme, dus git-pushes naar `uk` raken de live store niet meer.
