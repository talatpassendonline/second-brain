This folder is the operator's primary daily journal (professional mode). One file per day: `YYYY-MM-DD.md`.

## Rules

- [[me|Talat]] owns the daily; agents augment an existing daily, never invent one.
- Today's daily is today's activity only. Open loops from prior days live in `Resources/Operator/Tasks.md`, not re-appended here.
- The daily is state, not a log: update sections in place, no per-run timestamped callouts.

## Daily Note Frontmatter

```yaml
---
type: daily
date: YYYY-MM-DD
tags: [topic1, topic2]
---
```
